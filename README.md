# Информационная система "Библиотека"

## Вариант 4

Простая консольная информационная система для учета книг в библиотеке, написанная на Python.

### Функции:
*   Добавление книги в каталог.
*   Поиск книги по названию.
*   Выдача книги читателю.
*   Возврат книги в библиотеку.
*   Просмотр всего каталога.

### Используемые технологии:
*   Язык программирования: Python 3
*   Система контроля версий (локальная): Mercurial (hg)
*   Удаленный репозиторий: GitHub
